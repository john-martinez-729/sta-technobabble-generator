# sta-technobabble-generator
## 🖖 Star Trek Adventures 2e Technobabble Generator
A lightweight tool for Gamemasters and players to generate authentic sounding scientific jargon during Star Trek Adventures 2e sessions.
Whether you need to "recalibrate the graviton emitter" or "bypass the bio-neural gel pack," this generator uses the official randomized tables from the STA 2nd Edition Core Rulebook and Technical Manual to provide instant immersion.
## ✨ Features
STA 2e Compliant: Built using the specific prefix, subject, and verb tables found in the latest edition.
Context-Aware: Generate specific babble for Engineering, Science, or Medical scenarios.
Complication Generator: Includes 20+ random complication tables to spice up failed rolls.
Quick-Copy: One-click to copy the generated phrase directly into your virtual tabletop (VTT).
## 🚀 Getting Started
Clone the repo: git clone https://github.com
Run the app: Open index.html (or run python main.py)
Boldly Go: Hit the "Generate" button to solve your next warp core breach.
## 🛠 Usage Example
Select either Techno or Medical jargon from the slider. This will automatically generate babble using the tables found on pages 266 and 267 of the Star Trek Adventures 2e core rulebook.
You can re-roll individual values, or select them from directly. You can also hit the "Generate Babble" button to roll an entire new babble.
